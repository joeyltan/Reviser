# Reviser

Many commercial tools exist to support creative writers in their writing endeavors. However, research on current writing support applications and software has shown that there are few options that support structural revision of writing, editing on a text base level, or diversity of visualization in the writing process. For many creative writers, editing and revising are spatial and visual activities, yet they must resort to physical methods of printing and rearrangement without viable online methods. To address this, we develop ReViser, a creative writing software application for virtual reality, specifically on the Apple Vision Pro. It allows for taking a text document, such as one from Google Doc, Word, or other word processors, and being able to split the document into sections demarcated by the writer. The focus of the research is on spatial movement, manipulation, and capability of making connections between different layers of writing. 

Features that ReViser supports include:

Splitting a Google Doc / Microsoft Word / other text file into section indicated by the writer
	- Organizing split sections in space (where in a matrix-like layout or freehand)
	- Be able to make notes / comments on any section of a writer's choice
	- Be able to delete sections, move them around, swap with other sections in different spaces, or archive them into a ``graveyard"
	- Compare multiple draft versions side by side
	- Be able to splice back together the cut up manuscript back into a regular scrollable document  
	- Be able to tag sections based on a certain layer of connection (through some writing throughline element) and see / edit just a particular layer's sections, as well as view visually how they are linked
	- Include basic styling and text formatting capabilities, like bold, italics, strike through, indentation, font size, type, etc.
